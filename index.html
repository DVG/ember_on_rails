Ember.js on Rails

Prepared by Bradley Temple

Why Ember?

  • Clean separation of concerns between the front end behavior and the backend logic
  • Powerful data binding
  • Doesn't want you to forget perfectly good technologies like Jquery and HTML

Angular is clearly more popular, why waste my time?

  • Angular is cool, but it does encourage you to write invalid HTML, and in a project of size it's easy to lose various ng attributes in the shuffle

Getting started

              
gem 'ember-rails'              # for bootstrapping
gem 'ember-auth-rails'         # if you need authentication
gem 'active_model_serializers' # Helpful in rendering JSON
              
            
              
$ bundle install
$ rails g ember:bootstrap
